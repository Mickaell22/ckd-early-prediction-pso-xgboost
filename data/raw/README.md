# Dataset UCI Chronic Kidney Disease

## Español

El CSV crudo no se versiona en Git. Para obtenerlo:

1. Descargar el dataset desde el UCI Machine Learning Repository:
   https://archive.ics.uci.edu/dataset/336/chronic+kidney+disease
2. Colocar el archivo CSV limpio en este directorio como `ckd_uci.csv`.

El dataset contiene 400 instancias y 25 atributos, con aproximadamente 60%
de instancias con al menos un valor faltante. Licencia CC BY 4.0, donado por
L. Rubini, P. Soundarapandian y P. Eswaran (2015).

## English

The raw CSV is not versioned in Git. To obtain it:

1. Download the dataset from the UCI Machine Learning Repository:
   https://archive.ics.uci.edu/dataset/336/chronic+kidney+disease
2. Place the clean CSV file in this directory as `ckd_uci.csv`.

The dataset contains 400 instances and 25 attributes, with approximately 60%
of instances having at least one missing value. CC BY 4.0 license, donated by
L. Rubini, P. Soundarapandian and P. Eswaran (2015).
